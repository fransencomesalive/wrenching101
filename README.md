# Wrenching 101

Standalone Mettle Cycling microsite for `wrenching101.mettlecycling.com`.

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Asset Folders

- `public/fonts/` — licensed New Athletic 54, Gotham, and Patheos files
- `public/diagrams/` — source bike and component schematics
- `public/rive/` — exported Rive files
- `public/spline/` — Spline exports or scene notes

## Expected Font Filenames

The CSS is wired to these local files:

- `public/fonts/New Athletic M54.ttf`
- `public/fonts/Gotham-Light.otf`

- `public/fonts/Gotham-Bold.otf`
